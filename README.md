# Web-Hooker

### How to Install

Termux commands.

* `pkg update`

* `pkg upgrade`

* `pkg  install git`

* `pkg  install python3`

* `pkg  install wget`

* `pip install requests`

* `git clone https://github.com/kdo2064/Web-Hooker`

* `cd Web-Hooker`

* `python3 WebHooker.py`


Linux commands.

* `sudo apt update`

* `sudo apt upgrade`

* `sudo apt  install git`

* `sudo apt  install python3`

* `sudo apt  install wget`

* `pip3 install requests`

* `git clone https://github.com/kdo2064/Web-Hooker`

* `cd Web-Hooker`

* `python3 WebHooker.py`
